# Tianmeng Daily Summary

天蒙公司运营管理部日报提交系统，一个简洁高效的日常工作报告管理平台。

![Tianmeng Daily Summary](logo.png)

## 项目简介

Tianmeng Daily Summary 是一个专为天蒙公司运营管理部开发的日报提交系统。系统旨在简化日常工作报告流程，提高部门间信息传递效率，为管理层提供及时准确的数据支持。通过该系统，各部门可以快速提交每日工作报告，管理员可以实时监控提交情况，并进行数据分析和导出。

## 主要功能

### 1. 多部门日报提交
- 支持8个二级部门（景交车队、餐饮、商超、酒店、景点管理、游客中心、检票、保洁）
- 自定义日报模板，包含以下内容：
  - 今日主要工作
  - 发现及需要解决的问题
  - 明日工作计划
  - 数据统计
  - 备注信息
- 实时提交，即时保存
- 自动填充默认值，提高填写效率

### 2. 智能数据管理
- 自动记录提交时间和更新时间
- 完整的历史数据存档
- 支持多种格式数据导出：
  - Excel格式导出
  - JSON格式导出
- 数据备份机制

### 3. 管理员功能
- 部门提交状态实时监控
- 历史记录查询和检索
- 多日期数据查看
- 数据导出与分析
- 管理员密码保护
- **补交日报功能**：允许为未按时提交的部门进行补交操作
  - 自由选择部门和日期
  - 支持填写完整日报内容
  - 自动覆盖同日期已有数据
  - 实时更新提交状态和历史记录

## 使用说明

### 普通用户
1. 选择所属部门
2. 填写日报内容
3. 点击提交按钮
4. 等待提交成功提示

### 管理员
1. 点击"管理员登录"按钮
2. 输入管理员密码
3. 查看部门提交状态
4. 选择日期查看历史记录
5. 导出所需数据
6. **补交日报操作流程**：
   - 在管理员面板中找到"补交日报"区域
   - 选择需要补交的日期和部门
   - 点击"加载表单"按钮
   - 如果该日期该部门已有数据，将自动填充到表单中
   - 填写或修改日报内容
   - 点击"提交补交报告"按钮完成补交
   - 系统会自动更新提交状态和历史记录

## 常见问题

1. 数据如何备份？
   - 系统自动在backup目录创建备份
   - 建议定期手动备份data目录

2. 如何修改管理员密码？
   - 修改script.js中的adminPassword变量

3. 数据导出失败怎么办？
   - 检查浏览器兼容性
   - 确保有足够的存储空间
   - 联系管理员

4. 如何为错过提交日期的部门补交日报？
   - 管理员登录后，在管理面板中使用"补交日报"功能
   - 选择需要补交的具体日期和部门
   - 填写内容并提交即可
